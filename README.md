# Neural-Networks-for-the-Publishing-Industry
Neural Networks for the Publishing Industry:  Enhancing Customer Experience and Sales 

This project about neural networks to analyze and predict customer behavior, recommend 
books, and forecast demand in the publishing industry, improving both customer 
experience and sales. 

Approach: 
Data Understanding & Preprocessing,Exploratory Data Analysis (EDA) , Feature Engineering, Model Development , Model Evaluation.

Skills take away From this Project:

1.Data Cleaning and Preprocessing 
2.Exploratory Data Analysis (EDA) 
3.Feature Engineering for Neural Networks 
4.Building and Training Artificial Neural Networks (ANNs) 
5.Model Evaluation and Optimization 
6.MYSQL usage and integreation
7.Streamlit 

Results
A functional system for predictions, recommendations, and demand forecasting.
